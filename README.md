十指捺印卡模板与切割工具

简介
- 该项目生成一个十指捺印卡（模板），并提供预处理脚本用于从拍摄的卡片照片中检测卡片、做透视变换并切割出 10 个指印区域（crop）。

文件说明
- `src/generate_template.py`：生成 `template.png` 和 `template.json`（规范化的指区坐标）。
- `src/preprocess_card.py`：从输入照片检测卡片并输出切割后的 10 个指区图像文件。
- `run_demo.py`：演示流程：生成模板 -> 生成模拟拍照 -> 运行预处理 -> 输出 crops 到 `demo_output/`。
- `requirements.txt`：依赖列表。

安装

在 fish 或其他 shell 中运行：

```bash
python -m pip install -r requirements.txt
```

运行 demo：

```bash
python run_demo.py
```

示例输出
- `template.png`, `template.json`（模板）
- `simulated_capture.png`（模拟拍照）
- `demo_output/finger_1.png` ... `finger_10.png`（切割结果）

注意与改进方向
- 当前默认模板使用“角块辅助定位”：三角形 + 三个方块
  - 左上角是黑色三角形（非对称），其余三个角是黑色方块
  - 这样除了能做透视矫正（homography），还能消除 180° 上下颠倒的歧义：三角形出现在哪个角，就能知道卡片旋转了多少度
- 实战中可进一步增强鲁棒性：
  - 角块检测失败时回退到“最大四边形轮廓”定位（已内置）；
  - 对倾斜、部分遮挡、噪声做更强的预处理；
  - 支持批量处理（多图/多卡）。

高度定制复杂捺印卡（推荐流程）
1. 先确定“模板坐标系”：
  - 本项目把所有裁切框都定义在 `template.png` 的像素坐标系里，并在 `template.json` 里保存为归一化坐标（0~1）
  - 预处理阶段会把拍摄照片透视矫正到该模板坐标系，然后按框裁切

2. 你可以如何定制：
  - 改 `src/generate_template.py` 里的版式参数（页面尺寸、指区框数量/大小、间距、标签等）
  - 增加更多区域：例如“手掌印”“备注区”“条码区”“编号区”“签名区”等，只要把区域框加入 `template.json` 的 `boxes`（或扩展为多类 boxes）
  - 保留三角形角标：不要让 4 个角标完全同形，否则无法仅靠角标区分上下颠倒（180°）

3. 对非常复杂的卡：
  - 最稳的做法是先把卡片设计固定为一张“规范模板图”（SVG/AI/PS 都可以），然后把你要裁切的所有区域都标注为模板坐标
  - 我可以把生成器改成“读一个 JSON/CSV 配置 -> 自动生成 template.json（裁切框 + 标签 + marker 定义）”，这样你只需要维护配置文件，不用改代码
